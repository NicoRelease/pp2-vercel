🏗️ Arquitectura del Sistema

  • Arquitectura: MVC (Modelo-Vista-Controlador).
  • Backend: Node.js con Express, estructurado en capas de Controladores y Servicios.
  • Frontend: React.js con hooks para gestión de estado.
  • Persistencia: PostgreSQL mediante Sequelize ORM.
  • Seguridad: Autenticación mediante JSON Web Tokens (JWT).

💻 Guía de Instalación Local

1. Clonar el repositorio

git clone https://github.com/NicoRelease/pp2-vercel.git
         cd pp2-vercel

2. Instalación de Dependencias

Instale los módulos de Node necesarios para el proyecto:

Backend:

cd Backend
npm install

Frontend:

cd FrontEnd
npm install

3. Configuración de Variables de Entorno

Debe crear los archivos de configuración necesarios:

  • .env.local en la raíz del proyecto (contiene variables globales)
  • .env.local en la carpeta Backend/ (variables específicas de backend)
  • .env.local en la carpeta FrontEnd/ (variables específicas de frontend)

4. Ejecución en modo Desarrollo

Para iniciar el backend:

cd Backend
npm run dev

Para iniciar el frontend:

cd FrontEnd
npm run dev

🧪 Pruebas Unitarias

Para ejecutar la suite de tests y generar el reporte de cobertura en consola:

Backend:

cd Backend
npm test

Frontend:

cd FrontEnd
npm run test
